#  Isometric Dungeon Game
*No name yet*

With a friend, we are currently developing an isometric dungeon crawler built in Unity.

The game takes place inside a mysterious cave system where the player must fight through multiple floors to reach the surface.
*story etc in progress*

---

## Current Features

- Player movement with dash mechanic
- Combat system (work in progress)
  - Melee sword attacks (zone damage for the moment)
  - Projectile attacks
- Basic enemy AI
- Inventory system (UI implemented, no icons yet)
- Interaction system with objects
- Leveling system with scaling stats (HP / damage / XP)

---

##  Built With

- Unity (3D)
- C#
- Visual Studio
- Blendert (for the 3d models etc)
